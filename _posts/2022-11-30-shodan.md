# Shodan 'Cyber Explorer'
> recon via 3rd party 

[shodan](https://www.shodan.io/) is a powerful search engine to search for devices connnced to the internet from cameras to satellites. 

## Keyword
search filters are important it helps in narrowing down the search to very specifc things such as a satallite in the US. Keywords can be combined to form more complicated searches

```console
vsat country:US
```
**For router admin panel:**
```console
netgear 
```

### [Keyboard](https://www.shodan.io/search/filters)
Link that contains all the keyword/filter cheatsheet

## Features Offered
- [main](https://www.shodan.io/): search engine has limit for nonmembers
- [exploit](https://exploits.shodan.io/welcome): helps find vulnerable systems online
- monitor: track devices, , membership required
- maps: explore devices via map, membership required
- images: get and search screenshots, search images, membership required
